# Lynx Inverter Record
A simple application for recording and visualizing solar inverter data with InfluxDB, Telegraf, Grafana and Kapacitor.

## Supported Inverters

- Power-One, Inc. Aurora
    - TRIO-5.8/7.5/8.5-TL-OUTD
- Kostal
